# User Stories - TaskFlow API

## US-001: タスク作成

| 項目       | 内容                                                                                                           |
| ---------- | -------------------------------------------------------------------------------------------------------------- |
| ID         | US-001                                                                                                         |
| タイトル   | タスクを作成する                                                                                               |
| 優先度     | Must                                                                                                           |
| ストーリー | 開発メンバーとして、タイトル・説明・期限を入力してタスクを作成したい。それは自分の作業を記録し追跡するためだ。 |
| ソースFR   | FR-001                                                                                                         |

### 受け入れ基準

1. **Given** 認証済みのユーザーがいる, **When** タイトルと期限を指定してタスク作成APIを呼ぶ, **Then** タスクが作成されIDが返される
2. **Given** 認証済みのユーザーがいる, **When** タイトルなしでタスク作成APIを呼ぶ, **Then** 400エラーが返される

## US-002: ステータス変更

| 項目       | 内容                                                                                                     |
| ---------- | -------------------------------------------------------------------------------------------------------- |
| ID         | US-002                                                                                                   |
| タイトル   | タスクのステータスを変更する                                                                             |
| 優先度     | Must                                                                                                     |
| ストーリー | 開発メンバーとして、タスクのステータスをTodo/InProgress/Doneに変更したい。それは進捗を可視化するためだ。 |
| ソースFR   | FR-002                                                                                                   |

### 受け入れ基準

1. **Given** Todoステータスのタスクがある, **When** InProgressに変更する, **Then** ステータスが更新され更新日時が記録される
2. **Given** Doneステータスのタスクがある, **When** Todoに変更しようとする, **Then** 422エラーが返される

## US-003: タスク割り当て

| 項目       | 内容                                                                                         |
| ---------- | -------------------------------------------------------------------------------------------- |
| ID         | US-003                                                                                       |
| タイトル   | タスクをメンバーに割り当てる                                                                 |
| 優先度     | Must                                                                                         |
| ストーリー | 開発リーダーとして、タスクを特定のメンバーに割り当てたい。それは作業分担を明確にするためだ。 |
| ソースFR   | FR-003                                                                                       |

### 受け入れ基準

1. **Given** 開発リーダーがログインしている, **When** タスクにメンバーを割り当てる, **Then** タスクのassigneeが設定される
2. **Given** 開発メンバーがログインしている, **When** 他人のタスクを再割り当てしようとする, **Then** 403エラーが返される
